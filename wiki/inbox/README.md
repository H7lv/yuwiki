# 📥 收件箱

> 随手记的东西放这里，后面统一整理。

## 用法

- 手机、电脑上想到什么就丢到这里，一个文件一条笔记
- 不需要讲究格式，纯文本即可
- 文件名建议用日期+关键词，如 `2026-06-25-买车想法.md`
- 说"整理收件箱"或 `@/inbox`，我会帮你分类归档到对应页面

## 规则

- inbox 里的文件我会处理，处理完后会移到对应目录或归档
- 本文件（README.md）不会被处理
